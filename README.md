## 📋 Descrição do Projeto

Este projeto implementa um **pipeline completo de Machine Learning com MLOps** para o problema de **classificação binária** de depressão em adolescentes (`depression_label`), utilizando uma rede neural Multilayer Perceptron (MLP).

O pipeline contempla todas as etapas exigidas na avaliação:
- Limpeza e pré-processamento dos dados
- Seleção de variáveis com múltiplos métodos (Pearson, Mutual Information e Random Forest)
- Treinamento e comparação de diferentes arquiteturas de MLP
- Versionamento completo com **Weights & Biases (W&B)**
- Análise crítica dos resultados
